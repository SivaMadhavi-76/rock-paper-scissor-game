# 🪨 Rock Paper Scissors Game

This is a simple **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**.
It is a beginner-friendly project that I created as part of my **learning journey last year** to practice core web development concepts.

## 📌 About the Project

This project allows a user to play the classic Rock–Paper–Scissors game against the computer.
The computer randomly selects a choice, and the result is displayed instantly.

## 🚀 Features

* Interactive UI
* Random computer selection
* Score tracking
* Instant result display
* Simple and responsive design

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 🎯 Purpose

This project was built to:

* Practice basic HTML structure
* Learn CSS styling
* Understand JavaScript DOM manipulation
* Implement simple game logic



## 📚 Learning Outcome

Through this project, I learned how to:

* Handle user interactions
* Use JavaScript for logic building
* Manipulate DOM elements
* Create small interactive web applications
